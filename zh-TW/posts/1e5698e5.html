Web

題目名字不重要反正題挺簡單的

透過 /index.php?file=phpinfo 訪問 phpinfo，在其中可以找到 flag。

NewWebsite

隨手點選主頁文章，可以發現 cid 是數字索引，猜測存在 SQL 注入。

依照一般的注入方式得到帳號密碼。

在嘗試之下找到 /admin 路由然後使用注入得到的帳號 admin，密碼 admin 登入上管理面板。一番查詢之後可以找到 /admin/?r=manageinfo，發現其頁面下有一個檔案上傳。

嘗試上傳一句話木馬。

上傳 phtml，訪問對應的檔案 http://47.111.104.99:50800/upload/touxiang/99621604249634.phtml，發現上傳成功。使用蟻劍連上去，即可得到 flag。

Misc

顏文字之謎

用 Wireshark 分析流量包，可以發現 index-demo.html 的響應的 html 中有一段 base64，將整個 Line-based text data 按位元組流匯出。將 base64 片段摘取出來。

使用 base64 隱寫的指令碼可以跑出 key:"lorrie"。

使用 010editor 開啟檔案的時候不難發現最後有個可疑的空行，結合解密出的 key 和題目描述，不難想到 snow 隱寫。於是在 Kali Linux 下使用 stegsnow -p lorrie export.html export.txt 提取檔案。

使用記事本開啟後得到如下內容，將 →_→ 替換為 -，將 ←_← 替換為 .，空格替換為 / 後得到一段摩斯電碼。

解碼後得到 67B33E39B5105FB4A2953A0CE79C3378，將其轉為小寫之後得到 flag。

passwd

用 volatility 先確定檔案的 profile python vol.py -f WIN-BU6IJ7FI9RU-20190927-152050.raw imageinfo。

然後根據題目描述直接嘗試讀取使用者密碼 python vol.py -f WIN-BU6IJ7FI9RU-20190927-152050.raw --profile=Win7SP1x86 mimikatz。

將密碼 qwer1234 按照題目描述 sha1 處理之後得到 flag。

虛實之間

zip檔案要密碼，用 010editor 檢視，以為偽加密，改了 09 00 改成 00 00 結果發現好像不是偽加密，那就是真加密。想著 ARCHPR 可以直接爆破但是不行，字典也不行，最後考慮了一下 zip 壓縮包明文攻擊。

將 mingwen-副本.txt 檔案壓縮，檢視壓縮演算法，發現與原 zip 檔案中的 mingwen.txt 中相同，用 AZPR 爆破得到口令，解壓得到flag.txt。

這很像我很久之前分析過的蝦米音樂 API 裡的 location，先排成一個方陣然後再轉置即可，這裡補全 5 之後直接套用我很久以前寫的函式。

隱藏的祕密

用 volatility 先確定虛擬記憶體檔案的 profile python vol.py -f p.vmem imageinfo，然後使用 filescan 讀取其檔案列表，可以在其桌面下發現 file.txt。

然後提取這個檔案，使用文字編輯工具開啟，可以得到 hint。

然後，使用 registrydump 匯出所有登錄檔檔案。

使用 Eric R. Zimmerman 的 RegistryExplorer 載入 SAM 相關的登錄檔檔案，找到唯一一個登陸過的賬戶。

使用 hivelist 找出登錄檔，然後使用對應的鍵值和 hashdump 查出使用者密碼的 hash 值，篩選出對應的使用者，得到 FHREhpe$:70fdb8f853bd427d7584248b8d2c9f9e:f3cf477fc3ea6ec0b3b5887616dd4506:::。

將 hash 的後半部分 f3cf477fc3ea6ec0b3b5887616dd4506 放到 cmd5 網站上嘗試解密得到 NIAIWOMA。拼接上使用者名稱，得到 FHREhpe$:NIAIWOMA，將其按照 hint MD5 處理之後得到 flag。
