首先看一下 bashlex 的抽取邏輯，以 ls 這條指令為例。

CommandNode 裡包括 WordNode，每個 Node 包括 pos 和 parts。所以此時的規則大概是指令中不能含有變數、不能以路徑開頭、如果是純字母的指令則不允許白名單以外的指令。同時不可以使用 bash 的指令替代（但是這點有點迷惑）。所以只需要找到一個非純字母組成的指令即可。這裡因為是 python 的環境。於是有 python3 的指令可以被成功執行。由於沒有回顯，所以嘗試開個 http 服務。
