Change my mind

使用 Stegsolve 解隱寫可以發現 flag。

Forbidden

附件給出了一個 .car 檔案。使用 binwalk 可知其中包含 bzip2 的壓縮資料。

接著嘗試使用 binwalk -e 將檔案分離出來。分離出來後發現直接得到了 flag。

Granny wants you

Help my granny login, while monitoring my network one day I was able to capture the credentials, go through this file to get ther credentials

Flag format : username:password ( no – {} )

Author: dboidembla

這題給的流量包中有 300+ 個 TCP 流，因此題目描述中的 login 和 credentials 很重要。結合題目描述可知要找的是登入憑證。既然是登入那應該是 POST 請求。篩選出 POST 請求後可在其中一個裡找到如下資訊。

將其中的資訊摘取下來。

其中的 E0IPJIyDGxq2Mx56oz12LKD%3D 經過 URL Decode --> From Base64 --> ROT13 的路徑解碼後可得到 TROLLCATisAmazing。pass 引數的內容將 + 替換為空格後再經過 From Binary --> From Base64(N-ZA-Mn-za-m0-9+/=) 的路徑解碼後可以得到 Welcome_to_CSCodershub。（需要注意這一步的 base64 解碼是換表的解碼）根據題目描述將兩段字串用冒號拼接即可得到 flag。

Rich Orphan

參考：

https://passlib.readthedocs.io/en/stable/lib/passlib.hash.md5_crypt.html

根據附件給出的 hash 格式可知是 md5_crypt。

按照參考文件寫個指令碼使用 GitHub 一萬常見密碼錶爆破。

執行指令碼可以得到密碼是 batman。

Mr_evilpepo

volatility2 跑一下 imageinfo 得到 profile。

cmdline 下可以發現一個編輯 mysecret.txt 的指令，於是 filescan 一下將其在記憶體中的位置找出來。

使用 dumpfiles 將這個文件提取出來，可以得到以下內容。

將檔案下載下來，得到 secret 檔案。接著在映象中找線索，使用 cmdscan 檢視 cmd 的執行歷史，可以發現一個 flag。

在 pslist 中還可以發現 Chrome 的程序，於是提取 Chrome 的歷史記錄。

可以發現使用者訪問了 VeraCrypt 的下載頁面和一個加密 Pastebin 的網站。同時往上找可以得到使用者貼上的內容的網址。

使用 hashdump 讀取使用者賬戶密碼的 hash 2e6a7cf5aabb33a044684dd9c97e88a7。使用線上破解可以得到密碼為 abracadabra。

使用這個密碼解密加密的 Pastebin 可以得到如下內容。

使用 VeraCrypt 載入 secret 檔案，使用同一個密碼掛載磁碟。開啟掛載的磁碟可以得到 foryou.txt，其中包含第三個 flag。

s3cr3t

使用 FTK Imager 掛載 trollcat.E01 檔案。

在 X:\[root]\Favorites\drive 下可以找到 topsecret.zip 檔案。其中包含一個需要解鎖 bitlocker 的虛擬磁碟檔案。寫個指令碼使用 GitHub 上的一萬個常用密碼爆破 BitLocker 密碼。

掛載磁碟後執行指令碼可以得到如下執行結果。

指令碼在 52768.7111ms 後解鎖了磁碟。使用 FTK Imager 訪問掛載後的磁碟，可以在回收站中找到 flag 檔案。

the_sus_agent

Wireshark 分析流量包，在 TCP 流 0 處可以發現 welcome.jpg。在 TCP 流 88 處可以發現名為 secret 的字串。

將得到的字串 aWhvcGV5b3VkaWRub3R0cmllZHRvYnJ1dGVmb3JjZWl0 base64 解碼一次可得 ihopeyoudidnottriedtobruteforceit。將圖片提取出來後上傳到 aperisolve 使用這個字串作為密碼分析，可以得到以下結果。

將檔案下載下來解壓即可得到 flag。
